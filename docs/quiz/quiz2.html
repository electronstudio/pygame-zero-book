  1. Which of these programs is correct?

    1. print{"hello world"}

    2. print["hello_world"]

    3. print("hello world")

    4. print "hello world"

    5. print <"hello_world">

  2. What will this program output?

    x = 7
y = x+2
print("Answer:", y)

  3. What will this program output?

    x = 5
x = x*2
print("Answer:", x)

  4. This incorrect program has a syntax error. Write a correct version with no error.

    print("hello world)

  5. What will this program output?

    x = 12
x += 2
print("Answer:", x)

    2. output(“Dinner is ready”)

    3. say(“Dinner is ready”)

    4. ”Dinner is ready”
